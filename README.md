# dandelion

demoscene-style git repository visualizer. one HTML file, raw WebGL2

heavily inspired by [gource](https://gource.io), but now without needing a local binary

## how to use

1. run `./generate-log.sh` inside any git repo to produce a log file
2. open `gloam.html` in a browser (test scene loads by default)
3. drag the `.log` file onto the page to visualize

alternatively just type a public repo in for it to be cloned by the website
